## What is c++
is a programming language created by Bjarne Stroustrup in 1985. The language was designed with the following principle:

- Zero-cost abstraction: High-level abstractions should not incur additional runtime or memory overhead
- Full control: The programmer has complete control over hardware and system resources
- Safety: Enforce safety guarantees at compile time whenever possible
